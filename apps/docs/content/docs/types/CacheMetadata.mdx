## CacheMetadata

Configuration options for cache behavior

| Property | Type                                                                                              | Value | Description                                                             |
| -------- | ------------------------------------------------------------------------------------------------- | ----- | ----------------------------------------------------------------------- |
| name     | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | N/A   | Custom name for the cache entry                                         |
| ttl      | string \| number                                                                                  | N/A   | Time-to-live duration in milliseconds or as a string (e.g., '1h', '5m') |
